# Time to price convertor
Paste time list into left field, and get price list

### Example of time list

1. First task - 10 min
2. Second task - 20 min
3. Third task - 1 hour 30 min
4. Forth task - 1 hour 10 min
5. Fifth task - 5 hour 45 min

**Hour rate:** 20$

#### Results in:

1. First task - 3.33$
2. Second task - 6.66$
3. Third task - 30$
4. Forth task - 23.33$
5. Fifth task - 115$
